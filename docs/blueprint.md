# **App Name**: Al-Huda

## Core Features:

- Responsive UI: Create a responsive user interface optimized for both tablets and mobile phones, ensuring a consistent and pleasant experience across devices.
- Al-Quran Text Display: Display the Al-Quran text in multiple formats: Arabic only, Arabic-Indonesian transliteration, Arabic with Indonesian translation, and Arabic-Indonesian transliteration with Indonesian translation.
- Custom Firebase Database: Implement a custom database using Firebase with the name 'aplikasselamat' to store and manage Al-Quran data and user-related information.
- Kemenag Compliance: Ensure the Al-Quran text and translations comply with the standards and guidelines set by the Indonesian Ministry of Religious Affairs (Kemenag).
- AI-Powered Tafsir Tool: Implement a tool to search Al-Quran verses and associate them with appropriate Tafsir interpretations. Use the LLM to reason about how closely the Tafsir aligns to the verse being displayed to the user.

## Style Guidelines:

- Primary color: Soft light green (#C8E6C9) to create a peaceful and calming atmosphere.
- Background color: Very light green (#F1F8E9), a desaturated version of the primary color, for a gentle backdrop.
- Accent color: Slightly darker green (#A5D6A7) to highlight important elements and interactive components.
- Body and headline font: 'Alegreya', a serif font with an elegant, intellectual, contemporary feel, for both headlines and body text.
- Use simple, meaningful icons to enhance user navigation and comprehension.
- Implement a clean and intuitive layout that is easy to navigate and read on both tablets and mobile phones.
- Incorporate subtle 3D animations to add visual interest without overwhelming the user.
